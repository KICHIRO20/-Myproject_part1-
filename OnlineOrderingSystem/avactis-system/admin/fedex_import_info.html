FedEx rates importing information

FedEx rate estimation is based on rate tables, so before using rate estimation for FedEx you need to download rate tables from FedEx site and import them.

First you need to obtain the following files from FedEx:

  1. Download .csv files for the following services from ftp://ftp.fedex.com/pub/us/rates/downloads/documents2/
    or from
    http://www.fedex.com/ratetools/RateToolsMain.do

    2nd Day
    Express Saver
    First Overnight
    Priority Overnight
    Standard Overnight
    Ground
    Home Delivery
    International Economy
    International Priority
    International First
    Express: US to Puerto Rico

  2. Please make sure you have entered your origination Postcode at the "Store settings -> Store Owner's Profile" section. After that please download the ASCII zone chart at this page:

    http://www.fedex.com/ratetools/RateToolsMain.do

Put all these files on your server where Avactis is installed in the directory '<avactis-dir>/avactis-system/modules/shipping_module_fedex/imort_data/'. Then press 'Import' button at the FedEx configuration page. Please do not forget to update rate tables and 'Fuel surcharges' regulary to keep it up to date.
